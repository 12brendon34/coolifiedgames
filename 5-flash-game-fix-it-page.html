All of Coolmath's games and amusements are functioning perfectly. So, if the game isn't working for you, then there's a problem on your side that can probably be fixed.  ALL browsers (except one's from the early 90's -- haha) should be able to run these games.

If you do not see the word "Coolmath" bouncing around in the blue rectangle below,  you need to adjust your browser...

1) If you can see the bouncing "Coolmath" above and the games look like they are up and ready, but you can't get it to start, click your mouse on the game area. Sometimes the game just needs to know you are ready to go.  

2) If you can't see the bouncing "Coolmath" above, then you probably need to download and install (or update) a program called Flash.  You can download it here:  Adobe Flash Downloads

3) If you've downloaded and installed Flash and things still aren't working, then you need to adjust your browser.  If you are using Internet Explorer...  The very first thing is that, if your browser warns you that it's blocked some "active content" on Coolmath, go ahead and follow the directions to allow it.  If you didn't get that choice,

  • Go to Tools on the menu at the top of the browser...

  • Select Internet Options...

  • Click on the Advanced tab...

  • Scroll down and look for settings for "active content" and make sure they are set to allow...

  • Reboot the computer (if necessary) and everything should be fine.

4) If the above didn't help, then it's probably an issue with your antivirus program blocking the content.  See what you can do to change those settings.  Sorry, but those are all different, so I can't give any specifics.
